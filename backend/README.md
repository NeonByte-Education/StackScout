## Описание
 
Backend часть StackScout — это мощный микросервис на **Java Spring Boot**, отвечающий за сбор данных, анализ лицензий и расчет показателей "здоровья" библиотек.
 
## Возможности
 
### Сборщик данных (Data Collector)
- Автоматический сбор метаданных через официальные API (PyPI JSON API, Docker Registry API).
- Планируемый парсинг через Cron jobs.
- Параллельная обработка с использованием очередей сообщений.
 
### Модуль лицензионного соответствия
- Нормализация названий лицензий.
- Классификация рисков (Разрешающие, Copyleft, Проприетарные).
- Матрица совместимости.
 
### Алгоритм оценки здоровья
Интеллектуальная система оценки на основе актуальности релизов, активности разработки и поддержки сообществом.
 
### Защищенный API
- JWT-авторизация (Spring Security).
- RBAC (управление доступом на основе ролей).
- Rate Limiting.
- Спецификация OpenAPI 3.0 (Swagger).
 
## Технологический стек
 
- **Язык**: Java 21
- **Фреймворк**: Spring Boot 3.5.10
- **База данных**: PostgreSQL (основная), H2 (разработка)
- **Кэширование**: Redis
- **Очереди**: RabbitMQ
- **Безопасность**: Spring Security + JWT

## Быстрый старт
 
### Требования
- Java 21
- Docker (для PostgreSQL/Redis/RabbitMQ)
 
### Запуск в режиме разработки
 
```bash
./gradlew bootRun --args='--spring.profiles.active=dev'
```
По умолчанию приложение запустится на порту `8081` с использованием H2 в памяти.
 
### API Документация
Доступна после запуска по адресу:
**http://localhost:8081/swagger-ui.html**
 
### Тестирование
 
```bash
./gradlew test
```

## Архитектура Backend
 
| Компонент            | Технология                   | Назначение                                 |
| -------------------- | ---------------------------- | ------------------------------------------ |
| **API слой**         | Spring Boot + Spring Web     | RESTful endpoints с JWT авторизацией       |
| **Слой данных**      | PostgreSQL + Spring Data JPA | Постоянное хранилище и ORM                 |
| **Слой кэширования** | Redis + Spring Cache         | Управление сессиями и ограничение запросов |
| **Система очередей** | Spring AMQP + RabbitMQ       | Асинхронный парсинг и планирование         |
| **Мониторинг**       | Prometheus + Grafana         | Сбор метрик и визуализация                 |
